Open Source Reverse Proxy

ProxyGate

A lightweight, self-hosted reverse proxy with a built-in web UI for managing routes, TLS certificates, and DNS records.

Disclaimer: This software is provided for educational and development purposes. It has not been through a formal security audit. Do not use in production environments without thorough review.
View on GitHub Quick Start Guide

Features

Everything you need to proxy local services behind real domains with TLS.

Reverse Proxy

Route HTTP/HTTPS/WebSocket traffic by domain name to local services with automatic header forwarding and hot config reload.

Web Dashboard

Manage proxy routes, certificates, and DNS records through a clean, modern dark-themed admin UI.

Let's Encrypt

Request free TLS certificates via DNS-01 challenge with step-by-step guidance directly in the UI.

GoDaddy DNS

Create and manage DNS records directly from the UI using the GoDaddy API. Browse domains and records.

Auto Certificate Flow

One-click: create DNS record, verify challenge, and install certificate — fully automated.

Live Reload

WebSocket-powered real-time updates. Config changes apply instantly without restart.

How It Works

A single binary that runs three servers and manages everything through a unified config.

Quick Start

Get up and running in under a minute.

git clone https://github.com/gcis/proxygate.git
cd proxygate
make build
./bin/proxygate

# Admin UI at http://127.0.0.1:9090

# With network whitelist for admin UI:
./bin/proxygate --allow-network 192.168.1.0/24 --allow-network 10.0.0.0/8
